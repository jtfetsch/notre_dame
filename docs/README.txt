Required packages:
Redis v2.8.8 or higher (BUILT FROM SOURCE)
node v0.10.21 or higher
npm v1.4.6 or higher
MondgoDB v2.4.9 or higher
nginx v1.4.7 or higher

The following node libraries are used, so it is recommended that you read up on them to understand how this project works:
underscore <http://underscorejs.org>
mongoose <http://mongoosejs.com>
socket.io <http://socket.io>
express <http://expressjs.com>
ejs <http://embeddedjs.com>
mobile-detect <https://github.com/hgoebl/mobile-detect.js>
then-redis <https://github.com/mjijackson/then-redis>
q <https://github.com/kriskowal/q>
imap <https://github.com/mscdex/node-imap>

The following node packages are utilized in the build process. The associated tasks run in an automated fashion, but some familiarity with their usage is suggested:
grunt <http://gruntjs.com>
grunt-contrib-sass <https://github.com/gruntjs/grunt-contrib-sass>
grunt-contrib-jst <https://github.com/gruntjs/grunt-contrib-jst>
grunt-contrib-copy <https://github.com/gruntjs/grunt-contrib-copy>
grunt-contrib-requirejs <https://github.com/gruntjs/grunt-contrib-requirejs>
bower <http://bower.io>
forever <https://github.com/nodejitsu/forever>
